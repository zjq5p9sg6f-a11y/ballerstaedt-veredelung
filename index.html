Ballerstaedt Veredelungs-Mockup

K3-Plugin · Siegelfolien-Veredelung in 3D erlebbar
Phase 0 · live React 19 + R3F Module Federation BaCo Basics 3.0 konform

Form. Material. Logo. Prägung.
Live in 3D.

Kunde sieht seine Siegelfolie bevor sie gedruckt wird — Logo-Upload, Druck oder Blindprägung, metallische Heißfolie, parametrische Form. Eingebettet in den K3-Konfigurator von ObjectCode, entwickelt für Ballerstaedt & Co. OHG.

Dieses Plugin lädt sich als Module-Federation-Remote in einen K3-Konfigurator und liefert das 3D-Modell «Siegelfolie» sowie die Logo-Upload-Komponente. Standalone ist nur dieser Endpoint sichtbar — die Live-Interaktion findet im K3-Konfigurator statt.

Was das Plugin liefert

Dynamic Model

Type · sealFoilPlugin

Parametrische Siegelfolien-Geometrie:

  • Form: round oder oval
  • Durchmesser: 20–150 mm
  • Material: Aluminium glänzend/matt, Kunststoff
  • 9 BaCo-Form-Codes (R, AR, K, AK, AL, IR, PSL, …)

Variable Template

Logo / Druckbild · Type image

File-Upload-Komponente, akzeptiert PNG / JPG / SVG. Logo wird wahlweise als Druckbild (Diffuse-Map) oder als Blindprägung (Sobel-generierte Normal-Map) auf die Folie projiziert.

Veredelungs-Modi

12 BaCo-Prägungen + 4 Druck-Stile aus dem Werkshandbuch BaCo Basics 3.0 als Konfigurations-Schema:

  • Diamant grob/fein, Leinen-Damast, Würmchen, Pünktchen
  • Nadelstich, Schriftzug-Streu, Werkzeug-Logo, Erhabene Logo
  • Zieldruck, Streudruck, Einfärbung, Heißfolie
